学习汇编目的
        基于 8086CPU，通过学习关键指令来深刻理解机器运行程序的机理，从而随心随欲的控制计算机；
        充分获取对计算机底层的编程体验，培养底层编程意识和思想。

        最终运行程序的是 CPU，用汇编语言编程的时候，必须要从 CPU 的角度考虑问题。

汇编语言编程相关工具
        编辑器（Edit）
        编译器（MASM）
        连接器（link）
        调试工具（Debug）

汇编程序从写出到执行的过程
        编写（Edit，*.asm）
        编译（MASM，*obj）
        连接（link,*.exe）
        加载（由 Command 将 *.exe 加载到内存）
        运行（CPU 处理）
